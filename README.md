# Cowin Bot

Bot to notify users of open slots at given centres from cowin website

## Requirements

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install flask.

```bash
pip install flask
```

## Usage

```bash
python cowin_bot.py
```
